Les composants WeWeb

Vue d'ensemble des composants

WeWeb propose une bibliothèque complète de composants préconfigurés qui peuvent être utilisés pour construire n'importe quelle interface utilisateur. Ces composants sont organisés par catégories et entièrement personnalisables.

Catégories de composants

1. Composants de base

  • Container : Conteneur flexible pour organiser d'autres composants
  • Section : Zones principales de votre page
  • Text : Paragraphes, titres et texte formaté
  • Button : Boutons interactifs avec actions
  • Image : Affichage d'images optimisées

2. Composants de formulaire

  • Input : Champs de saisie de texte
  • Select : Listes déroulantes
  • Checkbox : Cases à cocher
  • Radio : Boutons radio
  • Date picker : Sélecteur de date

3. Composants de données

  • Table : Affichage de données tabulaires
  • List : Listes répétables liées à des données
  • Cards : Cartes de contenu dynamiques

Propriétés des composants

Chaque composant possède plusieurs types de propriétés configurables :

  • Propriétés de contenu : Texte, images, données affichées
  • Propriétés de style : Couleurs, marges, padding, typographie
  • Propriétés de comportement : Conditions d'affichage, interactions
  • Propriétés de données : Binding avec des variables et collections

Bonnes pratiques

  1. Réutilisabilité : Créez des composants personnalisés pour les éléments répétitifs
  2. Organisation : Nommez clairement vos composants et organisez-les logiquement
  3. Responsive : Testez vos composants sur différentes tailles d'écran
  4. Performance : Évitez l'imbrication excessive de composants

Exemple pratique : Créer une carte utilisateur

Pour créer une carte affichant les informations d'un utilisateur :

  1. Ajoutez un composant Container
  2. Insérez une Image pour la photo de profil
  3. Ajoutez des composants Text pour le nom et la description
  4. Liez les données à votre collection d'utilisateurs
  5. Stylisez le tout avec le panneau de propriétés
